# LP02B - Radar de Ingresos Landing Page

Esta es la landing page oficial de ventas para el producto **Radar de Ingresos**, diseñada con un enfoque absoluto en la conversión a través de tráfico directo.

## Características
- **Estética Premium:** Modo oscuro elegante con acentos en verde lima y tipografía Montserrat.
- **Interactividad en Vivo:** Incluye un widget de demostración donde el usuario simula su compatibilidad ingresando presupuesto y tiempo.
- **Carga Rápida:** Implementada con HTML5 semántico, CSS3 Vanilla y Javascript nativo sin frameworks pesados.
- **100% Responsivo:** Adaptación impecable en dispositivos móviles y de escritorio.

## Estructura del Código
- `index.html`: Estructura y copy de los 12 bloques.
- `styles.css`: Estilos visuales, layouts y animaciones sutiles.
- `script.js`: Lógica interactiva de sliders, FAQ acordeón y modal de checkout.
